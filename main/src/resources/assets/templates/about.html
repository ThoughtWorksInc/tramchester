Tramchester

About this app

The goal for ThoughtWorks Manchester Tech Lab was to experiment with cutting edge open source technology while trying to deliver a product for the local community in 6 weeks.

We used Neo4j Graph Database to model the tram data provided from Data GM. Graph databases are well-suited for the Travel and Transport Industry as they excel at mapping physical locations against the various ways that one might get people or things to and from those locations. Built-in graph algorithms enable extremely fast execution of common problems, such as “shortest-path”.

Tramchester is now maintained by ThoughtWorks as opensource software

Live data for the app comes from TFGM live data API.

We used Amazon Cloud Infrastructure. The benefits of this is the low starting cost, elastic scalability and easy to set up.

Continuous design and delivery was the backbone of this application. Using a mobile first response design approach, with lots of user involvement through out the process.

Other technology involved:

  • Java
  • Angular JS
  • Open street map

ThoughtWorks is a software company and community of passionate individuals whose purpose is to revolutionise software design, creation and delivery, while advocating for positive social change.
